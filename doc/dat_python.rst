==============
DAT for Python
==============

Overview
========

The primary Data Analytics Toolkit for Python is `xarray <http://xarray.pydata.org/en/stable/>_` since the central
data structure used for representing the ESDC in Python is the
`xarray.Dataset <http://xarray.pydata.org/en/stable/data-structures.html#dataset>_`.

In addition, some higher-level DAT functions are available
in the `cablab.dat <https://github.com/CAB-LAB/cablab-core/blob/master/cablab/dat.py>_` module.
